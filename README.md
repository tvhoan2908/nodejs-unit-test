## Description

- Project Unit Test For NodeJs.

## System Requirements

- NodeJs 12 or higher.
- Mysql 8
- Sequelize
- Jest

## Getting started

```bash
$ npm install
$ npm test
```

- Test e2e:

```bash
$ npm run test:e2e
```

- Test specific file:

```bash
$ npx jest file path
$ eg: npx jest tests/users/user.service.spec.js
```
